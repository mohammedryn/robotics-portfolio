Analog PID-Based Line Following Robot

This project is a line follower robot built using analog IR sensors and a PID (Proportional-Derivative) control algorithm to follow a black line on a white surface with high precision and smooth movement. The robot uses three analog IR sensors to detect the line position and adjusts its motor speeds accordingly using a PID algorithm implemented on an Arduino-based microcontroller. The motor control is achieved using the DRV8833 dual H-bridge motor driver.

‚öôÔ∏è Hardware Components

Arduino (e.g., Nano/Uno), 3x TCRT5000 analog IR sensors, DRV8833 motor driver, 2x N20 DC gear motors, Wheels and chassis, Battery pack (7.4V or similar).

üß† How It Works

  • Three analog IR sensors detect the line, giving values based on surface reflectivity.
  • A weighted average method computes the error in the robot's position.
  • The Proportional (Kp) and Derivative (Kd) components are used to calculate the correction factor.
  • The left and right motor speeds are adjusted based on the correction value to steer the robot.

‚úÖ Project Highlights

  • Implements PID control for smoother and accurate line following.
  • Works well on curved tracks due to proportional correction.
  • Ideal as a foundation for advanced robotics: maze solvers, obstacle avoiders, or autonomous path planners.
